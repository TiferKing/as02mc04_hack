# AS02MC04_hack

AS02MC04 板卡逆向资料

## 包含内容

 - constraints: 约束文件，描述引脚作用。
 - documents: 参考文档。
 - examples: 样例工程。
 - as02mc04: 板卡文件。

## 使用方法

将文件夹as02mc04复制进 Vivado/\<version\>/data/boards/board_files/ ，如果没有board_files目录就自行创建一个。

打开Vivado新建工程即可选择“AS02MC04 NIC Card”。
